CAT FIGHT!

Developed by Alex Nguyen & Kenny Crowell


Zip File | GitHub

Overview:

Cat Fight! is a mobile game optimized for iPhone6's resolution, but played on a personal computer. It is a fast-paced, two-player, puzzle game where players challenge their friends. On each player’s screen, there is a tile board, the player’s health bar, and the opponent’s health bar. The goal of the game is to get your opponent’s health down to zero. To do so, the player must make chains of adjacent tiles. The player that can make the longest tiles in the shortest amount of time will ultimately win.

How to Set-Up:

Download the zip file of the game through the link above. Extract the file and run the game through the game's executable.

How to Play:

  • Finding an Opponent: Tap on the start screen. This will initiate the match finder. Once a match is found, the game will start.
  • Making chains: Swipe across chains of tiles that are 3 or longer. Let go at the end of the chain to register it. Chains can only be made using adjacent tiles.
  • Healing: Make a chain that is 5 or longer.
  • Refreshing the Board: Press the refresh button located to the left of the health bars. This will freeze your board temporarily as a punishment
